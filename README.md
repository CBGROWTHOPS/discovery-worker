# Discovery Worker

Railway-hosted Express service for Reddit discovery.

## Railway project (already created via API)

- Project ID: `823f52ff-69ac-461e-aa86-005d90f10bec`
- Service ID: `8fe97614-521a-47c8-aa14-4e60eb3ff100`
- Environment: production (`cbd6257a-d964-42de-bc13-9b0a3dd42367`)

## Deploy to Railway

1. Link (one-time):  
   `railway link -p 823f52ff-69ac-461e-aa86-005d90f10bec -s 8fe97614-521a-47c8-aa14-4e60eb3ff100`
2. Deploy: `railway up`
3. Generate domain: `railway domain`
4. Add the URL to Vercel as `DISCOVERY_WORKER_URL`
